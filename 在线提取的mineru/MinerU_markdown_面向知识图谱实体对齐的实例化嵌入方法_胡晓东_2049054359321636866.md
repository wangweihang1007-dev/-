# 0引言

以不同方式构建的知识图谱[1](Knowledge Graph,KG)存在知识冗余和互补问题,随着图嵌入技术的不断发展,跨语言知识图谱的实体对齐[2](Entity Align-ment,EA)被广泛应用于知识转化、修复一致性冲突、构建跨语言程序等。实体对齐作为一种图嵌人技术,旨在发现不同知识图谱之间的等价实体对,是异构知识图谱链接的枢纽。

现有主流的实体对齐方法主要借助知识图谱的结构信息学习两个跨语言知识图谱的低维向量,通过嵌入向量相似性发现实体的邻接信息[3],从而生成实体对齐结果,分为浅层嵌入和深度嵌入。其中,浅层嵌入将知识图谱中的实体和关系投影到向量空间中,通过最小化全局对齐约束的损失函数进行训练,如MTransE[4]JAPE[5];深度嵌入由卷积网络(GraphConvolutionalNetwork,GCN)生成实体结构向量实现对齐结构,如GCN-Align[6]、GM-Align[7]。然而,在现实世界中要将不同知识图谱的表示向量映射到同一空间,需要足够的标注数据[8]作为图谱之间关系的链接,文献[9]采用迭代训练从测试集中遴选高置信度实体对用于下一轮训练,文献[10]提出在GCN基础上增加自注意力机制和图交叉注意力机制的MuGCN模型,文献[11]提出了BootStrapping 框架通过增加损失函数中负样本的影响,以提升实体对齐的准确率,但由于高置信度不一致和跨语言差异,这些方法只能引入少量高置信度实体对样本,未充分利用实体、关系和属性等实例信息的一致性约束,使跨语言图谱之间属性不平衡和模态不一致。文献[12]设计了一种采用字符嵌入模型挖掘属性值信息的框架,将不同知识图谱的实体向量映射到同一空间;文献[13]将实体结构信息与实体名相互补充,采用词移动距离以提升长尾实体对齐效果,但在这些方法中部分图谱缺少大量三元组的属性信息和关系信息,忽略了三元组中的属性值[14]

针对上述知识图谱实体对齐方法不足之处,提出一种实例化嵌入方法,本文方法结合图卷积网络和多阶关系嵌入捕获实体链接信息,构建基于关系和属性的跨语言实体对齐,为跨语言知识图谱融合提供支持。

# 1.1知识图谱实体对齐

将知识图谱定义为KG=(E,R,A,V,T,T),其中:E、R、A分别表示实体集、关系集和属性集,V表示值集合,TCE×R×E和TCE×A×V分别表示关系三元组集合、属性三元组集合。将源知识图谱表示为KG=(E,R,A,,V,,T,T),目标知识图谱表示为K,=(E,,R,,A,V,T,T),对于KG中的每个实体,目的是识别目标知识图KGi中的等价实体,生成对齐矩阵并输出对齐结果。

(1)生成对齐矩阵。生成表示Kc和Kc之间对齐结果矩阵S∈RIE,IxIE,|E|和|E|分别表示Kc和Kc的实体数量,每个对齐矩阵S中的组件S(e,e')表示实体e,∈E及其对应实体e'∈E之间标识匹配。

(2)输出对齐结果。假设实体对(es,e'),如果es和e'在对齐结果中匹配,则e∈E和e'∈E,作为一个链接进行对齐输出,最终目标是基于少量标注数据或在相应实体上没有任何实况信息的情况下识别潜在链接。

# 1.2框架设计

如图1所示,设计一个实例化嵌入框架。首先,从输入的知识图谱中提取关系网络并转换为多阶图卷积网络,将知识图谱实体嵌入低维向量空间,从基于实体嵌入的距离捕获两个实体的关系相关性。然后,利用学习嵌入检索进行关系对齐,并使用机器翻译策略进行属性对齐。最后,将不同视图进行组合,获得整体对齐结果。主要有4个功能:

(1)多阶关系网络嵌入。将实体的关系信息集成到框架中,从输入的知识图谱中提取关系网络;然后采用基于GCN学习使用网络实体关系嵌入,该嵌入由多个层组成,每个层按多个顺序编码形成网络拓扑,以对模态数据进行编码。同时设置了无监督损失函数,该函数等价于最小化违反一致性约束的情况。

(2)实体对齐。实体对齐反映了相同的现实世界实体,将其实体名进行等价计算,检查实体的一致性。

(3)关系对齐。用所有GCN层嵌入的关系信息计算对齐输出,在每一层构造单阶对齐矩阵;然后对矩阵进行加权和组合,以获得关系对齐输出。权重表示各层的重要性,在计算单阶矩阵之前,执行调整预处理并通过迭代衰减噪声的影响,检查关系一致性。

(4)属性对齐。基于实体的属性信息进一步固化候选链接,建立相应属性字典并采用Jaccard度量计算属性相似度,检查属性一致性。

# 5结语

本文提出一种面向知识图谱实体对齐的实例化嵌入方法,结合图卷积网络和多阶关系嵌入捕获实体链接信息,通过实例化嵌入增加对源知识图谱和目标知识图谱之间的关系差异和属性不匹配的鲁棒性,构建关系和属性的跨语言实体对齐,并通过无监督方法解决数据标注需求问题。后续工作将进一步研究知识融合降噪的问题,开发具有应用价值的高效知识融合系统,并应用于真实世界数据实例中。
